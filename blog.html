Important FAQ

What is the Document Object Model(DOM)?

It is a programming interface that allows us to create, change, or remove elements from the document.

How do you select an element from the DOM?

By using these: getElementsByClassName(), getElementById(), querySelector().

What is event delegation in the context of the DOM,and why is it useful?

Event delegation in JavaScript is a pattern that efficiently handles events. Events can be added to a parent element instead of adding to every single element.

How do you manipulate an element's attribute and styles using the DOM?

Using the createAttribute() method to add an attribute to an element node. And by using the style property.
